# Iolit Shop

A pwa mobile shop with lit, @alwatr

## Run Locally

Clone the project

```bash
  git clone https://github.com/MM25Zamanian/iolit-shop.git
```

Go to the project directory

```bash
  cd iolit-shop
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn watch
```

## Tech Stack

**Client:** Lit, Ionic, Alwatr

**Server:** Node

**Dev Ops:** Cloudflare
